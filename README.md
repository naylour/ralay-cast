# RelayCast

> Один источник. Много каналов. Полный контроль над публикацией.

RelayCast — персональный сервис для публикации контента сразу на несколько платформ из единого редактора. Пишешь один раз — сервис автоматически рассылает пост на все подключённые ресурсы: Telegram-каналы, группы, и другие платформы в будущем.

Изначально создан для использования расширенной Telegram-разметки (доступной только ботам), inline-кнопок и медиа — через собственный бот, без ограничений клиентского интерфейса.

---

## Возможности

- ✦ **Единый редактор** — пишешь пост один раз, выбираешь куда отправить
- ✦ **Мультиплатформенность** — подключай несколько Telegram-каналов, групп и других ресурсов
- ✦ **Расширенная разметка** — полная поддержка Telegram Bot API форматирования (MarkdownV2, HTML)
- ✦ **Inline-кнопки** — конструктор кнопок прямо в интерфейсе
- ✦ **Медиа** — фото, видео, документы с подписями
- ✦ **Предпросмотр** — видишь как будет выглядеть пост перед отправкой
- ✦ **Отложенная публикация** — планируй посты заранее
- ✦ **История публикаций** — лог всех отправленных постов по каналам

---

## Стек

- **Frontend:** SvelteKit / Svelte 5
- **Backend:** Elysia.js + Bun
- **База данных:** PostgreSQL + Prisma ORM
- **Публикация:** Telegram Bot API
