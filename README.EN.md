# RelayCast

> One source. Many channels. Full control over your publishing.

RelayCast is a personal content publishing service that lets you broadcast posts to multiple platforms from a single editor. Write once — RelayCast automatically delivers your post to all connected destinations: Telegram channels, groups, and more platforms in the future.

Originally built to leverage Telegram's extended message formatting (available only to bots), inline keyboards, and rich media — through your own bot, without the limitations of the standard client interface.

---

## Features

- ✦ **Unified editor** — write once, choose where to publish
- ✦ **Multi-platform** — connect multiple Telegram channels, groups, and other destinations
- ✦ **Extended markup** — full Telegram Bot API formatting support (MarkdownV2, HTML)
- ✦ **Inline keyboards** — build button layouts directly in the editor
- ✦ **Media support** — photos, videos, documents with captions
- ✦ **Live preview** — see exactly how the post will look before sending
- ✦ **Scheduled publishing** — plan posts in advance
- ✦ **Publishing history** — full log of sent posts across all channels

---

## Stack

- **Frontend:** SvelteKit / Svelte 5
- **Backend:** Elysia.js + Bun
- **Database:** PostgreSQL + Prisma ORM
- **Publishing:** Telegram Bot API
